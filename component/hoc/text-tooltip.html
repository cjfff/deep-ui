# props

Name Description Type Required Default
enableTooltip 是否启用 tooltip 不启用就把 content 作为 DOM 元素的 title 属性使用 Boolean false true
content 显示的内容，如果超出长度则会显示为省略号 String / Number false ""
className 额外注入 class String false -
line 限制内容为本应为多小行，超出显示省略号, 默认 -1，不启用，为正数则启用 Number false -1
