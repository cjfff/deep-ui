# loading-dialog

<template>
  <div>
    <loading-dialog
      title="标题"
      :visible.sync="visible"
      :confirm="handleConfirm"
    >
        <el-form-renderer :content="content" v-model="data">
        </el-form-renderer>
    </loading-dialog>
    <el-button @click="visible = true">打开弹窗</el-button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      visible: false,
      data: {},
      content: [
            {
                label: '用户名',
                id: 'username',
                type: 'input',
                el: {
                    type: 'username',
                    placeholder: '请输入密码'
                }
            },
            {
                label: '密码',
                id: 'password',
                type: 'input',
                el: {
                    type: 'password',
                    placeholder: '请输入密码'
                }
            }
        ]
    };
  },
  methods: {
    handleConfirm() {
      return new Promise(resolve => setTimeout(resolve, 2000));
    }
  }
};
</script>

# props

Name Description Type Required Default
visible 是否显示 dialog Boolean false false
confirmButtonText 确定按钮文字 String false 确定
cancelButtonText 取消按钮文字 String false 取消
hasFooter 是否有底部栏 Boolean false true
beforeClose 关闭前会执行检查是否可以关闭 Function false () => true
confirm 点击按钮触发的事件，传入对应的处理 promise 可以达到 loading 的效果 Function false () => true
confirmDisabled 禁用提交按钮 Function / Boolean false false

# slots

Name Description Default Slot Content
default - -
footer - -
footer-area - -

# events

Event Name Description Parameters
update:visible 双向绑定 visible -
close 关闭弹窗会执行 -
confirm confirm 事件完成时触发 -
