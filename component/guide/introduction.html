# 介绍

由于 vuepress 依赖的是 core-js 是 2.x 版本，而 vue-cli3 中依赖的是 3.x 版本.

所以写文档的时候，需要你去全局安装一下 vuepress 以及 core-js@2.6.11

npm i -g vuepress core-js@2.6.11
