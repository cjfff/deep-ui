# list-container

简单易用的形式实现 tab 点击切换效果，无论是配合动态组件渲染还是内容修改，都及其容易扩展!

# 基本使用

绑定的数据: /dashboard/edit/info
COPY 
<template>
    <div>
    <ClientOnly>
      <list-container v-model="value" :tabs="routes" @change="handleChange"/>
    </ClientOnly>  
      <div>
        绑定的数据:
        {{value}}
      </div>
    </div>
</template>

<script>
export default {
  data() {
    return {
      value: "/dashboard/edit/info",
      routes: [
        {
          label: '基本信息',
          value: '/dashboard/edit/info',
        },
        {
          label: '数据配置',
          value: '/dashboard/edit/data',
        },
        {
          label: '权限管理',
          value: '/dashboard/edit/permission',
        },
      ],
    };
  },
  methods: {
    handleChange(value) {
      console.log(value)
    }
  }
};
</script>

# props

Name Description Type Required Default
tabs example [ { value: 0, label: '这是第一项'}, { value: 1, label: '这是第二项'} ] Array false []
animation 是否有动画效果 Boolean false true
value - Number / String false -

Name Description Default Slot Content
